🔗 Conexión Firebase (dos computadoras)

Pega aquí tu objeto firebaseConfig para sincronizar votos, nombres y fotos entre ambas computadoras en tiempo real.

Sin configurar
Pasos rápidos:
1. console.firebase.google.com → Crear proyecto → Realtime Database → modo Test.
2. Configuración del proyecto → icono </> → registrar app → copiar el objeto firebaseConfig.
3. En Realtime Database → Reglas → pegar {"rules":{".read":true,".write":true}} → Publicar.
